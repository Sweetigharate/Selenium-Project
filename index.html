// Normal Chrome
WebDriverManager.chromedriver().setup();
WebDriver driver = new ChromeDriver();

// Headless (for CI/CD)
ChromeOptions opts = new ChromeOptions();
opts.addArguments("--headless", "--window-size=1920,1080", "--disable-gpu");
WebDriver driver = new ChromeDriver(opts);

// Incognito
opts.addArguments("--incognito");

// Common driver commands
driver.get("https://www.jiomart.com");     // open URL
driver.manage().window().maximize();       // maximise
driver.getTitle();                          // page title
driver.getCurrentUrl();                     // current URL
driver.navigate().back();                   // browser back
driver.navigate().refresh();               // refresh
driver.quit();                               // close + end session
Always call driver.quit() in a @AfterMethod or finally block — never skip it. Leaving Chrome processes open crashes CI pipelines and wastes memory.
🔍 Locators

All Selenium Locators — Java

All 8 strategies, XPath deep dive, CSS cheat sheet, and priority guide.

All 8 Strategies
// 1. ID — fastest, always first choice
driver.findElement(By.id("search-bar"));

// 2. Name — great for form fields
driver.findElement(By.name("mobileNumber"));

// 3. CSS Selector — fast, flexible
driver.findElement(By.cssSelector("button[data-testid='add-to-cart']"));

// 4. XPath — when CSS not enough
driver.findElement(By.xpath("//button[text()='Add to Cart']"));

// 5. Class Name
driver.findElement(By.className("success-toast"));

// 6. Tag Name
driver.findElement(By.tagName("h1"));

// 7. Link Text (exact)
driver.findElement(By.linkText("View Cart"));

// 8. Partial Link Text
driver.findElement(By.partialLinkText("View"));

// Find MULTIPLE elements → returns List
List<WebElement> cards =
  driver.findElements(By.cssSelector(".product-card"));
System.out.println("Count: " + cards.size());
XPath — All Patterns
// Relative (always use — not absolute)
//input[@id='search-bar']
//button[@type='submit']
//input[@placeholder='Enter mobile number']

// By text
//button[text()='Add to Cart']
//span[contains(text(),'₹')]

// Contains class
//div[contains(@class,'product-card')]

// Parent → child
//div[@class='cart-item']//button[@class='remove-btn']

// Following sibling
//label[text()='Mobile']/following-sibling::input

// Index (1-based!)
(//div[@class='product-card'])[1]      // first
(//div[@class='product-card'])[last()]  // last

// AND / OR
//input[@type='text' and @name='mobile']
//button[@id='ok' or @name='ok']

// Ancestor
//span[text()='Remove']//ancestor::div[@class='cart-row']
CSS Selector Cheat Sheet
#search-bar                   // by ID
.product-card                  // by class
input[type='text']             // by attribute
input[placeholder*='mobile']   // contains
input[placeholder^='Enter']    // starts with
input[placeholder$='number']   // ends with
div.cart-item > button.remove  // direct child
.product-card:nth-child(1)     // first
.product-card:last-child       // last
button[data-testid='checkout'] // data attribute ⭐ best practice

1stidID exists and is uniqueFastest

⏳ Waits

All 3 Types of Selenium Waits

Implicit, Explicit, Fluent — when to use each, all ExpectedConditions, and a reusable wait utility.

1. Implicit Wait

Global timeout — set once

// Set globally — applies to ALL findElement calls
driver.manage().timeouts()
      .implicitlyWait(Duration.ofSeconds(10));
Simple, one line Can mask real timing bugs Never mix with Explicit Wait
2. Explicit Wait — Always Use This

WebDriverWait + ExpectedConditions — all types

WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(15));

// Clickable
wait.until(ExpectedConditions.elementToBeClickable(By.id("add-to-cart"))).click();

// Visible
WebElement toast = wait.until(
  ExpectedConditions.visibilityOfElementLocated(By.className("toast")));

// Present in DOM (not necessarily visible)
wait.until(ExpectedConditions.presenceOfElementLocated(By.id("order-id")));

// Invisible (wait for loader to disappear)
wait.until(ExpectedConditions.invisibilityOfElementLocated(By.id("spinner")));

// URL contains
wait.until(ExpectedConditions.urlContains("checkout"));

// Exact text in element
wait.until(ExpectedConditions.textToBe(By.id("cart-count"), "1"));

// Title contains
wait.until(ExpectedConditions.titleContains("JioMart"));

// Alert present
wait.until(ExpectedConditions.alertIsPresent());

// Number of elements
wait.until(ExpectedConditions.numberOfElementsToBeMoreThan(
  By.cssSelector(".product-card"), 0));
3. Fluent Wait — Most Powerful

Custom polling interval + ignore exceptions

import org.openqa.selenium.support.ui.FluentWait;
import org.openqa.selenium.NoSuchElementException;

FluentWait<WebDriver> fluentWait = new FluentWait<>(driver)
  .withTimeout(Duration.ofSeconds(20))
  .pollingEvery(Duration.ofMillis(500))  // check every 0.5s
  .ignoring(NoSuchElementException.class);

WebElement el = fluentWait.until(d ->
  d.findElement(By.id("dynamic-content")));

// Custom condition — wait until cart count is NOT zero
fluentWait.until(d -> {
  String count = d.findElement(By.id("cart-count")).getText();
  return !count.equals("0");
});
Reusable WaitHelper Utility

utils/WaitHelper.java — Use in all page classes

public class WaitHelper {
  private WebDriverWait wait;

  public WaitHelper(WebDriver driver) {
    this.wait = new WebDriverWait(driver,
      Duration.ofSeconds(15));
  }

  public WebElement waitForClickable(By by) {
    return wait.until(ExpectedConditions.elementToBeClickable(by));
  }

  public WebElement waitForVisible(By by) {
    return wait.until(ExpectedConditions.visibilityOfElementLocated(by));
  }

  public void waitForInvisible(By by) {
    wait.until(ExpectedConditions.invisibilityOfElementLocated(by));
  }

  public void waitForUrl(String urlPart) {
    wait.until(ExpectedConditions.urlContains(urlPart));
  }

  public void waitForText(By by, String text) {
    wait.until(ExpectedConditions.textToBe(by, text));
  }
}

base/BasePage.java
package base;
import org.openqa.selenium.*;
import org.openqa.selenium.support.PageFactory;
import org.openqa.selenium.support.ui.*;
import java.time.Duration;

public class BasePage {
  protected WebDriver driver;
  protected WebDriverWait wait;

  public BasePage(WebDriver driver) {
    this.driver = driver;
    this.wait   = new WebDriverWait(driver,
      Duration.ofSeconds(15));
    PageFactory.initElements(driver, this);
  }

  protected void click(WebElement el) {
    wait.until(ExpectedConditions
      .elementToBeClickable(el)).click();
  }

  protected void type(WebElement el, String text) {
    wait.until(ExpectedConditions
      .visibilityOf(el));
    el.clear();
    el.sendKeys(text);
  }

  protected String getText(WebElement el) {
    wait.until(ExpectedConditions.visibilityOf(el));
    return el.getText();
  }

  protected boolean isDisplayed(WebElement el) {
    try { return el.isDisplayed(); }
    catch (NoSuchElementException e) { return false; }
  }

  protected void waitForUrl(String urlPart) {
    wait.until(ExpectedConditions.urlContains(urlPart));
  }

  protected void scrollToElement(WebElement el) {
    ((JavascriptExecutor) driver)
      .executeScript("arguments[0].scrollIntoView(true);", el);
  }
}

package pages;
import base.BasePage;
import org.openqa.selenium.*;
import org.openqa.selenium.support.FindBy;

public class LoginPage extends BasePage {

  @FindBy(name = "mobileNumber")
  private WebElement mobileInput;

  @FindBy(name = "otp")
  private WebElement otpInput;

  @FindBy(xpath = "//button[contains(.,'Get OTP')]")
  private WebElement getOtpBtn;

  @FindBy(xpath = "//button[contains(.,'Verify')]")
  private WebElement verifyBtn;

  @FindBy(id = "login-error-msg")
  private WebElement errorMsg;

  public LoginPage(WebDriver driver) { super(driver); }

  public void navigateTo() {
    driver.get("https://sit.jiomart.com/login");
  }

  public void login(String mobile, String otp) {
    type(mobileInput, mobile);
    click(getOtpBtn);
    type(otpInput, otp);
    click(verifyBtn);
    waitForUrl("home");
  }

  public String getErrorMessage() {
    return getText(errorMsg);
  }

  public boolean isErrorDisplayed() {
    return isDisplayed(errorMsg);
  }
}

package pages;
import base.BasePage;
import org.openqa.selenium.*;
import org.openqa.selenium.support.FindBy;
import java.util.List;

public class HomePage extends BasePage {

  @FindBy(id = "search-bar")
  private WebElement searchBox;

  @FindBy(cssSelector = ".product-card")
  private List<WebElement> productCards;

  @FindBy(id = "cart-count")
  private WebElement cartCount;

  @FindBy(id = "cart-icon")
  private WebElement cartIcon;

  public HomePage(WebDriver driver) { super(driver); }

  public void searchProduct(String query) {
    type(searchBox, query);
    searchBox.sendKeys(Keys.ENTER);
  }

  public void clickFirstProduct() {
    wait.until(ExpectedConditions
      .numberOfElementsToBeMoreThan(
        By.cssSelector(".product-card"), 0));
    click(productCards.get(0));
  }

  public String getCartCount() {
    return getText(cartCount);
  }

  public void goToCart() { click(cartIcon); }
}

package pages;
import base.BasePage;
import org.openqa.selenium.*;
import org.openqa.selenium.support.FindBy;

public class ProductPage extends BasePage {

  @FindBy(id = "product-title")
  private WebElement productTitle;

  @FindBy(cssSelector = ".product-price")
  private WebElement productPrice;

  @FindBy(id = "add-to-cart-btn")
  private WebElement addToCartBtn;

  @FindBy(className = "success-toast")
  private WebElement successToast;

  @FindBy(xpath = "//span[contains(@class,'out-of-stock')]")
  private WebElement outOfStockMsg;

  public ProductPage(WebDriver driver) { super(driver); }

  public String getProductName()  { return getText(productTitle); }
  public String getProductPrice() { return getText(productPrice); }

  public void addToCart() {
    scrollToElement(addToCartBtn);
    click(addToCartBtn);
    wait.until(ExpectedConditions.visibilityOf(successToast));
  }

  public boolean isAddToCartEnabled() {
    return addToCartBtn.isEnabled();
  }

  public boolean isOutOfStock() {
    return isDisplayed(outOfStockMsg);
  }
}

package pages;

public class CartPage extends BasePage {

  private List<WebElement> cartItems;

  public void navigateTo() {
    driver.get("https://sit.jiomart.com/cart");

  public String getTotalPrice() { return getText(totalPrice); }

  public boolean isCartEmpty() { return isDisplayed(emptyMsg); }

  public void proceedToCheckout() {
    click(checkoutBtn);
    waitForUrl("checkout");
  }

    driver.findElement(By.xpath(
      "//div[@class='cart-item'][contains(.,'" + productName +
      "')]//button[@class='remove-btn']")).click();
  }

package tests;
import pages.*;
import utils.TestData;
import org.testng.Assert;
import org.testng.annotations.Test;

public class OrderPlacementTest extends BaseTest {

  @Test(description = "Full COD order placement E2E")
  public void testCODOrderPlacement() {

    // ── STEP 1: Login ────────────────────────────────────
    LoginPage loginPage = new LoginPage(driver);
    loginPage.navigateTo();
    loginPage.login(TestData.MOBILE, TestData.OTP);
    Assert.assertTrue(
      driver.getCurrentUrl().contains("home"),
      "Login failed — not on home page");
    System.out.println("✅ Step 1: Login successful");

    // ── STEP 2: Search Product ────────────────────────────
    HomePage homePage = new HomePage(driver);
    homePage.searchProduct(TestData.SKU);
    System.out.println("✅ Step 2: Search results loaded");

    // ── STEP 3: Open Product ──────────────────────────────
    homePage.clickFirstProduct();
    ProductPage productPage = new ProductPage(driver);
    String productName = productPage.getProductName();
    String productPrice = productPage.getProductPrice();
    Assert.assertFalse(productName.isEmpty(), "Product name empty");
    Assert.assertTrue(productPage.isAddToCartEnabled(), "Button disabled");
    System.out.println("✅ Step 3: Product — " + productName + " @ " + productPrice);

    // ── STEP 4: Add to Cart ───────────────────────────────
    productPage.addToCart();
    Assert.assertEquals(
      homePage.getCartCount(), "1", "Cart count mismatch");
    System.out.println("✅ Step 4: Added to cart");

    // ── STEP 5: Verify Cart ───────────────────────────────
    CartPage cartPage = new CartPage(driver);
    cartPage.navigateTo();
    Assert.assertFalse(cartPage.isCartEmpty(), "Cart is empty!");
    Assert.assertEquals(cartPage.getItemCount(), 1);
    System.out.println("✅ Step 5: Cart verified — Total: " + cartPage.getTotalPrice());

    // ── STEP 6: Checkout ──────────────────────────────────
    cartPage.proceedToCheckout();
    CheckoutPage checkoutPage = new CheckoutPage(driver);
    checkoutPage.selectAddress();
    System.out.println("✅ Step 6: Address selected");

    // ── STEP 7: Select COD ────────────────────────────────
    checkoutPage.selectCOD();
    Assert.assertTrue(checkoutPage.isCodSelected(), "COD not selected");
    System.out.println("✅ Step 7: COD selected");

    // ── STEP 8: Place Order + Assert ──────────────────────
    checkoutPage.placeOrder();
    Assert.assertTrue(
      checkoutPage.getSuccessMessage().contains("Order Placed"),
      "Order success message not found");
    String orderId = checkoutPage.getOrderId();
    System.out.println("✅ Step 8: Order ID — " + orderId);
    Assert.assertNotNull(orderId, "Order ID is null");
  }

  @Test(description = "Out of stock product — button disabled")
  public void testOutOfStockProduct() {
    LoginPage lp = new LoginPage(driver);
    lp.navigateTo(); lp.login(TestData.MOBILE, TestData.OTP);
    driver.get("https://sit.jiomart.com/product/JM-OUT-001");
    ProductPage pp = new ProductPage(driver);
    Assert.assertTrue(pp.isOutOfStock(), "Out of stock msg missing");
    Assert.assertFalse(pp.isAddToCartEnabled(), "Button should be disabled");
  }

  @Test(description = "Empty cart — checkout button not visible")
  public void testEmptyCart() {
    LoginPage lp = new LoginPage(driver);
    lp.navigateTo(); lp.login(TestData.MOBILE, TestData.OTP);
    CartPage cp = new CartPage(driver);
    cp.navigateTo();
    Assert.assertTrue(cp.isCartEmpty(), "Cart should be empty");
  }
}

import org.openqa.selenium.interactions.Actions;

Actions actions = new Actions(driver);

// Right-click (context menu)
actions.contextClick(element).perform();

// Double-click
actions.doubleClick(element).perform();

// Ctrl+A (select all)
actions.keyDown(Keys.CONTROL)
       .sendKeys("a")
       .keyUp(Keys.CONTROL)
       .perform();

// Move to element and click (better than direct click)
actions.moveToElement(addToCartBtn).click().perform();

// Scroll to element by offset
actions.scrollToElement(element).perform();
JavaScript Executor

When Selenium clicks fail — use JS

JavascriptExecutor js = (JavascriptExecutor) driver;

// Click hidden or overlapping element
js.executeScript("arguments[0].click();", btn);

// Scroll to element
js.executeScript("arguments[0].scrollIntoView(true);", el);

// Scroll to bottom of page
js.executeScript("window.scrollTo(0, document.body.scrollHeight);");

// Scroll to top
js.executeScript("window.scrollTo(0, 0);");

// Get page title via JS
String title = (String) js.executeScript("return document.title;");

// Set value in a field (when sendKeys fails)
js.executeScript("arguments[0].value='9876543210';", mobileInput);

// Get inner text
String txt = (String) js.executeScript(
  "return arguments[0].innerText;", element);

Screenshot on Failure

utils/ScreenshotUtil.java

import org.openqa.selenium.TakesScreenshot;
import org.openqa.selenium.OutputType;
import org.apache.commons.io.FileUtils;
import java.io.File; import java.time.LocalDateTime;

public class ScreenshotUtil {
  public static String takeScreenshot(WebDriver driver, String name) {
    try {
      File src = ((TakesScreenshot) driver)
        .getScreenshotAs(OutputType.FILE);
      String path = "screenshots/" + name + "_"
        + LocalDateTime.now() + ".png";
      FileUtils.copyFile(src, new File(path));
      return path;
    } catch (Exception e) { return null; }
  }
}

All Annotations
AnnotationWhen It RunsUse For
@BeforeSuiteOnce before entire suiteDB/server setup
@AfterSuiteOnce after entire suiteClean up resources
@BeforeClassOnce before each test classClass-level driver setup
@AfterClassOnce after each test classClass-level teardown
@BeforeMethodBefore EACH test methodOpen browser, login

@BeforeGroupsBefore a group runsGroup-specific setup
@TestThe actual testTest logic + assertions
@Test Parameters
// Priority — lower number runs first
@Test(priority = 1)
public void loginTest() {}

// Groups — smoke, regression, sanity
@Test(groups = {"smoke", "regression"})
public void addToCartTest() {}

// Depends on another test passing first
@Test(dependsOnMethods = {"loginTest"})

import org.testng.annotations.DataProvider;

@DataProvider(name = "loginData")
public Object[][] getLoginData() {
  return new Object[][] {
    { "9876543210", "1234", true  }, // valid
    { "0000000000", "0000", false }, // invalid mobile
    { "9876543210", "9999", false }, // wrong OTP
  };
}

// Use it in test
@Test(dataProvider = "loginData")
public void testLogin(String mobile, String otp, boolean expectSuccess) {
  LoginPage lp = new LoginPage(driver);
  lp.navigateTo();
  lp.login(mobile, otp);
  if (expectSuccess) {
    Assert.assertTrue(driver.getCurrentUrl().contains("home"));
  } else {
    Assert.assertTrue(lp.isErrorDisplayed());
  }
}

Assert Methods Cheat Sheet
import org.testng.Assert;

Assert.assertEquals(actual, expected);          // exact match
Assert.assertNotEquals(actual, notExpected);     // not equal
Assert.assertTrue(condition, "msg if fail");    // must be true
Assert.assertFalse(condition, "msg if fail");   // must be false
Assert.assertNotNull(object);                    // not null
Assert.assertNull(object);                       // is null
Assert.fail("Force fail with message");          // always fail

// Soft Assert — run ALL assertions even if one fails
import org.testng.asserts.SoftAssert;
SoftAssert soft = new SoftAssert();
soft.assertEquals(cartCount, "1");
soft.assertTrue(successMsg.isDisplayed());
soft.assertAll(); // MUST call this at end!
Use SoftAssert when you want to verify multiple things on one page and see ALL failures at once — not just the first one. Perfect for checkout page verification.
